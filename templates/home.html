How This Project Helps

Early screening matters because CKD can progress silently. This major project combines user input, preprocessing, and a trained model to support initial risk awareness for patients and learners.

Data-Driven Input

Each field maps to parameters used during model training, including blood chemistry and symptom markers.

Transparent Parameters

Every input in the form is labeled with expected range or allowed encoded value for easier and safer entry.

Fast Risk Output
