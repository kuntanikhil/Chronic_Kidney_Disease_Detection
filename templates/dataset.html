Dataset Used: UCI Chronic Kidney Disease (CKD)

This project uses the CKD dataset from the UCI Machine Learning Repository. It is a structured medical dataset designed for classification of chronic kidney disease risk using routine clinical parameters. The summary below is based on UCI dataset metadata and variable documentation.

Features

Classification

Class Labels

ckd / notckd

Why This Dataset Was Chosen

The UCI CKD dataset is appropriate for this major project because it contains clinically meaningful variables used in nephrology-related screening and supports supervised learning for binary prediction. It includes both numeric and categorical fields, making it practical for demonstrating preprocessing, encoding, model training, and deployment in a real web workflow.

  • Captures blood chemistry, urine indicators, and comorbidity markers.
  • Matches real-world medical tabular data conditions with missing values.
  • Widely referenced educational benchmark for CKD classification tasks.

How It Is Used in This Project

  • Selected feature subset is aligned with form inputs used on the screening page.
  • Categorical attributes are encoded into numeric representations before model fitting.
  • Missing and inconsistent entries are handled during preprocessing before training.
  • Final model output is exposed through Flask for interactive risk prediction.
